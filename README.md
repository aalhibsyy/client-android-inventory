# client-android-inventory
	Inventory sederhana dengan transaksi dan data barang menggunakan Retrofit2 RxJava dan konsep MVP 

# Package yang di gunakan
	- Retrofit2 
	- RxJava 
	- Butterknife
	- Glide
	- Material Dialog

# Fitur 
	- CRUD 
	- Login (JWT)
	- Search 
	- Endless Scroll (Hanya di implemtasi pada modul supplier)
